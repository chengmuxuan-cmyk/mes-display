# mes-display

MES 展示端前端项目，基于 `Vue 3 + TypeScript + Vite + Element Plus + ECharts` 构建，用于生产制造相关数据看板、展示页或展示终端。

## 技术栈

- Vue 3
- TypeScript
- Vite 7
- Element Plus
- ECharts
- Pinia
- Vue Router
- Axios
- viewerjs / v-viewer

## 本地开发

```bash
npm install
npm run dev
```

## 构建命令

```bash
npm run build
npm run preview
```

## 项目结构

- `src/`: 页面、组件、路由和状态管理
- `public/`: 静态资源
- `.env*`: 环境配置
- `vite.config.ts`: 构建配置
